# Performance Dashboard Pro

Live Demo: https://performance-dashboard-pro-vrnb.vercel.app

Stress Test: https://performance-dashboard-pro-vrnb.vercel.app/stress


## Features
- Real-time Line, Bar, Scatter, Heatmap (Canvas)
- FPS + Frame time + Heap (MB) live stats
- Data controls (points, interval)
- Stress Test mode (`/stress`) with CSV export

## Tech
- Next.js 14 (App Router, TS)
- Canvas rendering hooks
- Lightweight store/hooks for data stream

## Run locally
```bash
npm install
npm run dev
# http://localhost:3000
